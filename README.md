Conversational IQ — DISC Assessment
Herramienta de evaluación de personalidad basada en el modelo DISC, desarrollada por Umbra Consulting. Diseñada para coaches ejecutivos y líderes que quieren entender su estilo de comunicación y el de sus equipos.

¿Qué es Conversational IQ?
Un test de 28 preguntas que identifica el perfil de personalidad DISC del participante y lo mapea a uno de 16 arquetipos únicos en español — con nombres diseñados para resonar culturalmente en México y LATAM.
El resultado incluye:

Arquetipo de personalidad con descripción profunda
Gráfico radar con puntuación por dimensión (D, I, S, C)
Áreas de fortaleza
Puntos ciegos
Qué energiza y qué drena al participante
Cómo se relaciona con otros estilos
Frase memorable que define el arquetipo

Lógica de arquetipos
El modelo DISC evalúa cuatro dimensiones: D (Dominance), I (Influence), S (Steadiness), C (Conscientiousness).
Los 16 arquetipos se determinan por la combinación y la intensidad relativa de las dos dimensiones con mayor puntuación:

Mayúscula sola → dimensión dominante pura (ej. D, I)
Mayúscula + minúscula → dominante fuerte con secundario presente (ej. Di, Dc)
Dos mayúsculas → ambas dimensiones igualmente fuertes (ej. DI, SC)

Cuando dos dimensiones tienen fuerza similar, el resultado incluye una nota explicando qué significa tener un perfil dual y cuál dimensión suele emerger según el contexto.
